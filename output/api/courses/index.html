Courses

You can retrieve information about courses for assignation through this endpoint.

NOTE: Courses are assigned to users through the Inventories endpoint, not the courses endpoint.

Methods

Index

https://learninglogin.com/api/courses

To retrieve the list of courses, send a GET request to the URL above, no extra parameters are required. Results are paginated following the RFC-8288 standard for web linking; links are provided in the response headers so they can be navigated programmaticaly.

Pagination response headers

Link                <https://learninglogin.com/api/courses.json?page=1>; rel="first",
<https://learninglogin.com/api/courses.json?page=1>; rel="prev",
<https://learninglogin.com/api/courses.json?page=4>; rel="last",
<https://learninglogin.com/api/courses.json?page=3>; rel="next"
Per-Page 64
Total 195
